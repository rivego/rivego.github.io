CF558E A Simple Task(线段树)

2021年08月18日 1.2k 字 大概 6 分钟

Codeforces Round #312 (Div. 2) Problem E

一道线段树的套路题（区间排序），没记错的今年蓝桥省赛就有这么一题，去年校赛也有一题，都是这一个套路。

​ A Simple Task

This task is very simple. Given a string S of length n and q queries each query is on the format i j k which means sort the substring consisting of the characters from i to j in non-decreasing order if k = 1 or in non-increasing order if k = 0.

Output the final string after applying the queries.

Input

The first line will contain two integers n, q (1 ≤ n ≤ 105, 0 ≤ q ≤ 50 000), the length of the string and the number of queries respectively.

Next line contains a string $S$ itself. It contains only lowercase English letters.

Next q lines will contain three integers each i, j, k (1 ≤ i ≤ j ≤ n, $k\in\left \{0,1 \right \} $).

Output

Output one line, the string S after applying the queries.

题意：

给你一个字符串与 $q$ 次操作，每次操作选择一个区间，$q$ 为1则将这个区间升序排列，$q$ 为0则将这个区间降序排列，输出最后的字符串。

思路：

这道题其实就是区间赋值的板子，改一改就能过了 （然而有废物调了一天 我不说是谁）

可以先写一下这一题Problem - 1698 Just a Hook（区间赋值板子）

可以先建立26个线段树来分别表示每个区间内某字母的个数，如 $tree[1][1]$ 表示根节点为 $1$ 的区间内字母 $a$​​​ 的个数。建树完成之后进行修改操作，假如当前操作为升序排列，可以先进行一次计数排序记录下当前区间的各字母个数，然后将给定区间的字母个数清零，再通过计数排序从左至右，从小到大将记录好的字母放入区间内，反之只要从大到小放置即可，具体操作见Code。
